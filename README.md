## 贪吃蛇多人联机版

### 技术栈

vue3+pinia+typescript+socket.io

### 项目启动

#### 客户端：

进入snake-socket-vue目录

执行以下命令

```shell
npm install
npm run dev
```

#### 服务端：

进入server-socket-node目录

执行以下命令(需要node环境)

```shell
npm install
node server.js
```



### 项目构思

前端vue3单页面项目，服务器node提供socket服务，实现多人房间式贪吃蛇
